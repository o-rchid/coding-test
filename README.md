# 시간복잡도
- 코딩 테스트에선 최악일 때의 연산 횟수를 나타내는 빅-오 표기법을 사용한다.
- 연산 횟수 = 알고리즘 시간 복잡도 X 데이터의 크기
- 시간 복잡도 도출 기준
  -   상수는 시간 복잡도 계산에서 제외
  -   가장 많이 중첩된 박복문의 수행 횟수가 시간 복잡도의 기준
